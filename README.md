## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This project requires **Node.js** environment and **npm** package manager.

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/JoraBarjomi/nms-front
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server
   ```sh
   npm run dev
   ```
